默认情况下，当项从一个容器（如文件夹或日历）移动到另一个容器时，此值会更改。 若要更改此行为，请使用 `Prefer: IdType="ImmutableId"`头。 有关详细信息，请参阅[获取 Outlook 资源的不可变标识符](/graph/outlook-immutable-id)。
