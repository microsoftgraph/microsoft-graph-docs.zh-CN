以下特殊文件夹名称可用于 OneDrive 和 OneDrive for Business。

| Name        | 文件夹 ID    | 说明                                                              |
|:------------|:-------------|:-------------------------------------------------------------------------|
| Documents   | `documents`  | “文档”文件夹。                                                    |
| Photos      | `photos`     | “照片”文件夹。                                                       |
| Camera Roll | `cameraroll` | “本机照片备份”文件夹。                                           |
| App Root    | `approot`    | 应用程序的个人文件夹。 通常在 中 `/Apps/{Application Name}` 。 |
| Music       | `music`      | “音乐”文件夹。                                                        |
| 录制  | `recordings` | "录制"文件夹。 此选项仅在 OneDrive for Business 和 SharePoint Online 中可用。 |
