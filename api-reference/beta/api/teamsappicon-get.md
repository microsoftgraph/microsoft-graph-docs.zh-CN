# <a name="get-teamsappicon"></a>获取 teamsAppIcon

命名空间：microsoft.graph

[!INCLUDE [beta-disclaimer](../../includes/beta-disclaimer.md)]

检索与[应用](../resources/teamsapp.md)的特定[定义](../resources/teamsappdefinition.md)关联的 [Teams 应用图标](../resources/teamsappicon.md)。

## <a name="permissions"></a>权限

需要以下权限之一才能调用此 API。要了解包括如何选择权限的详细信息，请参阅[权限](/graph/permissions-reference)。

| 权限类型                        | 权限（从最低特权到最高特权）                      |
| :------------------------------------- | :--------------------------------------------------------------- |
| 委派（工作或学校帐户）     | AppCatalog.Read.All、AppCatalog.ReadWrite.All、AppCatalog.Submit |
| 委派（个人 Microsoft 帐户） | 不支持。                                                   |
| 应用                            | AppCatalog.Read.All、AppCatalog.ReadWrite.All                    |

## <a name="http-request"></a>HTTP 请求

**获取 Teams 应用定义的颜色图标**

<!-- { "blockType": "ignored" } -->
```http
GET /appCatalogs/teamsApps/{teams-app-id}/appDefinitions/{app-definition-id}/colorIcon
```

**获取 Teams 应用定义的大纲图标**

<!-- { "blockType": "ignored" } -->
```http
GET /appCatalogs/teamsApps/{teams-app-id}/appDefinitions/{app-definition-id}/outlineIcon
```

## <a name="optional-query-parameters"></a>可选的查询参数

此操作支持 `$select` [OData 查询参数](/graph/query-parameters)和 `$expand` OData 查询参数来自定义响应。

## <a name="request-headers"></a>请求头

| 标头           | 值                      |
| :--------------- | :------------------------- |
| Authorization    | Bearer {token}。必需。  |

## <a name="request-body"></a>请求正文

请勿提供此方法的请求正文。

## <a name="response"></a>响应

如果成功，此方法在响应正文中返回 `200 OK` 响应代码和 [teamsAppIcon](../resources/teamsappicon.md) 对象。
