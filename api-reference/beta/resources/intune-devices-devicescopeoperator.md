# <a name="devicescopeoperator-enum-type"></a>deviceScopeOperator 枚举类型

命名空间：microsoft.graph

> **重要：**/beta 版本下的 Microsoft Graph API 可能会发生更改;不支持生产使用。

> **注意：** 适用于 Intune 的 Microsoft Graph API 需要适用于租户的 [活动 Intune 许可证](https://go.microsoft.com/fwlink/?linkid=839381)。

设备范围配置查询运算符。 可能的值为：equals、notEquals、contains、notContains、greaterThan、lessThan。 默认值：等于。

## <a name="members"></a>成员
|成员|值|说明|
|:---|:---|:---|
|无|0|未为设备范围配置设置运算符。|
|等于|1|用于 (等于) 的设备配置查询的运算符。|
|unknownFutureValue|2|未来扩展枚举（如 notEquals）的占位符值包含的不是Contains、greaterThan、lessThan。|
