# <a name="meetingspeaker-resource-type"></a>meetingSpeaker 资源类型

命名空间：microsoft.graph

[!INCLUDE [beta-disclaimer](../../includes/beta-disclaimer.md)]

表示已启用注册的联机会议扬声器。

## <a name="properties"></a>属性

| 属性 | 类型 | 说明 |
| :------- | :--- | :---------- |
| bio | String | 扬声器的 Bio。 |
| displayName | String | 扬声器的显示名称。 |
