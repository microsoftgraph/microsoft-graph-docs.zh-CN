# <a name="devicescopeparameter-enum-type"></a>deviceScopeParameter 枚举类型

命名空间：microsoft.graph

> **重要：**/beta 版本下的 Microsoft Graph API 可能会发生更改;不支持生产使用。

> **注意：** 适用于 Intune 的 Microsoft Graph API 需要适用于租户的 [活动 Intune 许可证](https://go.microsoft.com/fwlink/?linkid=839381)。

设备范围配置参数。 将来将使用它来添加更多参数。 Eg：设备范围参数可以是 OS 版本、磁盘类型、设备制造商、设备模型或范围标记。 默认值：scopeTag。

## <a name="members"></a>成员
|成员|值|说明|
|:---|:---|:---|
|无|0|未设置设备范围参数|
|scopeTag|1|将作用域标记用作设备范围配置的参数。|
|unknownFutureValue|2|将来扩展的占位符值。|
