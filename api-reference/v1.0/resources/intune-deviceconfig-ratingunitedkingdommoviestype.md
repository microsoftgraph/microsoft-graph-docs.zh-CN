英国电影评级标签

## <a name="members"></a>成员
|成员|值|说明|
|:---|:---|:---|
|allAllowed|0|默认值，允许所有电影内容|

|agesAbove12Cinema|6 |12A，适合 12 年以上电影版本|
|agesAbove15|7 |15，仅适用于 15 年及更旧的|
|一些|8 |仅适用于成人|
