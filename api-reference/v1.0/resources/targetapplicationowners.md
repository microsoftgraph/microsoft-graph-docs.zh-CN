# <a name="targetapplicationowners-resource-type"></a>targetApplicationOwners 资源类型

命名空间：microsoft.graph

在访问包分配策略中，此类型继承自 [subjectSet](../resources/subjectset.md) ，并指示应用程序所有者可以代表该应用程序请求访问包。


## <a name="properties"></a>属性

无。
## <a name="relationships"></a>关系
无。
