# <a name="root-resource-type"></a>根资源类型

命名空间：microsoft.graph

**根** Facet 指示对象是它的层次结构中最顶层的对象。
若此 Facet 有值（非 null），表明对象是根。
若值为 null（或缺失），表明对象不是根。

**注意**：尽管此 Facet 暂为空，但在今后推出的 API 版本中将在此 Facet 中填充其他属性。

## <a name="properties"></a>属性
