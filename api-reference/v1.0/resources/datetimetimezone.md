# <a name="datetimetimezone-resource-type"></a>dateTimeTimeZone 资源类型

介绍某个时间点的日期、时间和时区。

## <a name="properties"></a>属性
| 属性     | 类型   |说明|
|:---------------|:--------|:----------|
|DateTime|String|日期和时间组合表示形式的单个时间点 (`<date>T<time>`)。|
|TimeZone|String|以下时区名称之一。|
