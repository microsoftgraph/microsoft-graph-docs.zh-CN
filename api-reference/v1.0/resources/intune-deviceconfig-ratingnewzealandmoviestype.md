新西兰的电影分级标签

## <a name="members"></a>成员
|成员|值|说明|
|:---|:---|:---|
|allAllowed|0|默认值，允许所有电影内容|
|allBlocked|1|不允许任何电影内容|
|常规|2|适用于普通受众|
|parentalGuidance|3|PG 分类建议家长指导|
|成熟|4|M 分类适用于成熟的受众|
|agesAbove13|5|R13 分类仅限于 13 岁及以上的人|
|agesAbove15|6 |R15 分类仅限于 15 岁及以上的人|
|agesAbove16|7 |R16 分类仅限于 16 岁及以上的人|
|agesAbove18|8 |R18 分类仅限于 18 岁及以上的人|
|限制|9 |R 分类仅限于特定受众|
|agesAbove16Restricted|10|RP16 分类要求 16 岁以下的查看者由父母或成人陪同|
