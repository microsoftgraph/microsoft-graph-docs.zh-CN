# <a name="hyperlinkorpicturecolumn-resource-type"></a>hyperlinkOrPictureColumn 资源类型

命名空间：microsoft.graph

表示超链接或图片栏中SharePoint。

## <a name="properties"></a>属性

| 属性      | 类型               | Description|
|:-------------------|:-------------------|:----------------------------------------------|
| isPicture       | Boolean             | 指定用于 URL 列的显示格式是图像还是超链接。 |
