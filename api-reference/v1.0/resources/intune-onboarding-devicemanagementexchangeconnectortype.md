|成员|值|说明|
|:---|:---|:---|
|onPremises|0|连接到本地 Exchange 环境。|
|托管|1|连接到 O365 多租户 Exchange 环境|
|serviceToService|2|Intune 服务直接连接到 O365 多租户 Exchange 环境|
|专用|3|连接到 O365 专用 Exchange 环境。|
