加拿大的电视内容分级标签

## <a name="members"></a>成员
|成员|值|说明|
|:---|:---|:---|
|allAllowed|0|默认值，允许所有电视节目内容|
|allBlocked|1|不允许任何电视节目内容|
|children|2|C 分类适用于 2 至 7 岁儿童|
|childrenAbove8|3|C8 分类适用于 8 岁以上的儿童|
|常规|4|G 分类适用于普通受众|
|parentalGuidance|5|PG，家长指导|
|agesAbove14|6 |14+ 分类适用于 14 岁及以上的查看者|
|agesAbove18|7 |18+ 分类适用于 18 岁及以上的查看者|
