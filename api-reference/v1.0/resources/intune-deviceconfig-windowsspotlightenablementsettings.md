# <a name="windowsspotlightenablementsettings-enum-type"></a>windowsSpotlightEnablementSettings 枚举类型

命名空间：microsoft.graph

> **注意：** 适用于 Intune 的 Microsoft Graph API 需要适用于租户的 [活动 Intune 许可证](https://go.microsoft.com/fwlink/?linkid=839381)。

允许 IT 管理员为MDM-Controlled设备设置预定义的默认搜索引擎

|成员|值|说明|

|notConfigured|0|未配置锁屏上的聚焦|
|禁用|1|在锁屏上禁用 Windows 聚焦|
|enabled|2|在锁屏上启用 Windows 聚焦|
