## <a name="sharee-get-a-shared-calendar-or-its-events-directly-from-calendar-owners-mailbox"></a><span data-ttu-id="a7ea3-111">共享者：直接从日历所有者的邮箱获取共享日历或其事件</span><span class="sxs-lookup"><span data-stu-id="a7ea3-111">Sharee: Get a shared calendar or its events directly from calendar owner's mailbox</span></span>

<span data-ttu-id="a7ea3-112">以下三个示例假设这样的情景：在 Outlook 中，Alex 向 Megan 共享了他的主日历，并给予 Megan 读取权限。</span><span class="sxs-lookup"><span data-stu-id="a7ea3-112">The three examples below use this scenario: in Outlook, Alex has shared his primary calendar with Megan and given Megan read permissions.</span></span> <span data-ttu-id="a7ea3-113">如果 Megan 登录你的应用，并提供 _委托权限_ (Calendars.Read.Shared or Calendars.ReadWrite.Shared)，代表 Megan，你的应用可以直接从 Alex 的邮箱中访问 Alex 的主日历及其事件。</span><span class="sxs-lookup"><span data-stu-id="a7ea3-113">If Megan signs into your app and provides _delegated permissions_ (Calendars.Read.Shared or Calendars.ReadWrite.Shared), on behalf of Megan, your app can access Alex' primary calendar and its events directly from Alex' mailbox.</span></span>

<span data-ttu-id="a7ea3-114">三个示例指定了所有者的标识（Alex 的用户 ID 或用户主体名称）和 `calendar` 快捷方式。</span><span class="sxs-lookup"><span data-stu-id="a7ea3-114">The three examples specify the owner's identity (Alex' user ID or user principal name) and the `calendar` shortcut.</span></span> <span data-ttu-id="a7ea3-115">这些示例仅访问与所有者的邮箱对应的日历和事件 ID。</span><span class="sxs-lookup"><span data-stu-id="a7ea3-115">They access calendar and event IDs that correspond to only the owner's mailbox.</span></span> <span data-ttu-id="a7ea3-116">在共享者的邮箱（Megan 的用户 ID 或用户主体名称）中指定这些日历和事件 ID 将返回错误。</span><span class="sxs-lookup"><span data-stu-id="a7ea3-116">Specifying these calendar and event IDs in the sharee's mailbox (Megan's user ID or user principal name) would return an error.</span></span> <span data-ttu-id="a7ea3-117">若要使用与共享者的邮箱对应的日历和事件 ID，请参阅[共享者：从共享者的邮箱获取共享的自定义日历或其事件](#sharee-get-shared-custom-calendar-or-its-events-from-sharees-mailbox)。</span><span class="sxs-lookup"><span data-stu-id="a7ea3-117">To use calendar and event IDs that correspond to the sharee's mailbox, see [Sharee: Get shared, custom calendar or its events from sharee's mailbox](#sharee-get-shared-custom-calendar-or-its-events-from-sharees-mailbox).</span></span> 

> <span data-ttu-id="a7ea3-118">**注意**：通过共享权限（Calendars.Read.Shared 或 Calendars.ReadWrite.Shared），你可以在共享或委托的日历中读取或写入事件。</span><span class="sxs-lookup"><span data-stu-id="a7ea3-118">**Note**: The sharing permissions (Calendars.Read.Shared or Calendars.ReadWrite.Shared) allow you to read or write events in a shared or delegated calendar.</span></span> <span data-ttu-id="a7ea3-119">它们不支持[订阅此类文件夹中的项的更改通知](webhooks.md)。</span><span class="sxs-lookup"><span data-stu-id="a7ea3-119">They do not support [subscribing to change notifications](webhooks.md) on items in such folders.</span></span> <span data-ttu-id="a7ea3-120">若要对租户中共享、委托或任何其他用户或资源日历中的事件设置更改通知订阅，请使用应用程序权限 Calendars.Read。</span><span class="sxs-lookup"><span data-stu-id="a7ea3-120">To set up change notification subscriptions on events in a shared, delegated, or any other user or resource calendar in the tenant, use the application permission, Calendars.Read.</span></span>
